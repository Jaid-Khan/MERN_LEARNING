02 - Components and Props

Purpose:
This folder covers the concept of React Components and Props.
It explains how to create reusable components, pass data between them, and use props destructuring or children props.

Concepts Learned:

Components:

Functional Components are simple JavaScript functions that return JSX.

They help in breaking the UI into reusable, independent pieces.

Props (Properties):

Props are used to pass data from a parent component to a child component.

Props are read-only, meaning the child component cannot modify them.

Example: <Card name="Tony" age="20" gender="Male" />

Props Destructuring:

Instead of writing props.name, props.age, props.gender,
we can directly destructure props in the function parameters.

Example: function Card({ name, age, gender }) {}

Children Prop:

Anything written between <Card> ... </Card> is available as props.children.

Example: <Card>This is a Card.</Card>

Inside the component, it can be used as {children}.
