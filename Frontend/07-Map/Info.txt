This React component demonstrates how to dynamically render a table using an array of JavaScript objects. 
It uses the `useState` hook to manage the data and JSX to create the table structure.

âš™ï¸ 2. useState Hook
--------------------
The `useState` hook initializes a state variable called `arr` that holds an array of hero objects.
Each object represents a hero with the properties `name`, `age`, and `gender`.

```js
const [arr, newArr] = useState([
  { name: "Tony", age: 27, gender: "male" },
  { name: "Captain America", age: 27, gender: "male" },
  { name: "Hulk", age: 28, gender: "male" },
  { name: "Thor", age: 26, gender: "male" },
]);
```

- `arr`: current state value (array of heroes)
- `newArr`: function to update the array (currently unused)

ğŸ§  3. JSX Return Structure
---------------------------
The component returns a fragment (`<> </>`) that contains a table.

### ğŸ§± Table Structure
- `<thead>` â†’ contains the table headers
- `<tbody>` â†’ contains all table rows

```jsx
<table border={2} width={500}>
  <thead>
    <tr>
      {Object.keys(arr[0]).map((key, index) => (
        <th key={index}>{key}</th>
      ))}
    </tr>
  </thead>
  <tbody>
    {arr.map((value, i) => (
      <tr key={i}>
        {Object.values(value).map((val, j) => (
          <td key={j}>{val}</td>
        ))}
      </tr>
    ))}
  </tbody>
</table>
```

---

ğŸ” 4. Logic Explanation
------------------------
- `Object.keys(arr[0])`: extracts property names (e.g. `["name", "age", "gender"]`) for table headers.
- `arr.map(...)`: loops through each object in the array to create a row (`<tr>`).
- `Object.values(value)`: extracts property values (e.g. `["Tony", 27, "male"]`) for table cells (`<td>`).
- `key={index}`: provides a unique key to each element to prevent React warnings.
