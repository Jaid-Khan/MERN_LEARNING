Axios

A promise-based HTTP client used in React to send API requests to the backend.

Installed using:

Connecting to Backend

The React app (running on http://localhost:5173) makes a request to the backend server (running on http://localhost:3000).

Axios is used to call backend routes such as /, /api, etc.
