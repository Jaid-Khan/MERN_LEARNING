ğŸŒ 3. Fetching Data with Axios
-------------------------------
Axios is used to send a **GET request** to the backend server.

```js
axios.get("http://localhost:3000/students?s=ankit").then((res) => {
  console.log(res.data);  // logs response data in console
  setA(res.data);         // stores data in state
});
```

### âš ï¸ Important Note:
This request should be placed inside a **useEffect()** hook to avoid infinite loops, since `axios.get()` runs on every re-render.

Example (Best Practice):
```js
useEffect(() => {
  axios.get("http://localhost:3000/students?s=ankit")
       .then((res) => setA(res.data));
}, []);
```
