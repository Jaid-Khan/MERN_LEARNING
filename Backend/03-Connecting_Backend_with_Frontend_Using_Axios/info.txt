ðŸ§  Concepts Learned

Express

A minimal Node.js framework for creating servers and APIs.

Simplifies route creation and request handling.

CORS (Cross-Origin Resource Sharing)

Allows requests between different ports or domains.

Required because React (5173) and Express (3000) run separately.

Installed using:

npm install cors


Basic Routing

Routes define how the server responds to various URLs (e.g., /, /about, /api).

âš™ï¸ Setup Steps

File: server.js or index.js

const express = require("express");
const server = express();
const cors = require("cors");

// Enable CORS so frontend can access backend
server.use(cors());

// Create a simple GET route
server.get("/", (req, res) => {
  res.send("This is Home");
});

// Start server on port 3000
server.listen(3000, () => {
  console.log("Server listening on port 3000");
});
